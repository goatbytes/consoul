**Summary of `README.md`**

The `README.md` is the entrypoint documentation for the **Consoul** project, an AIpowered terminal assistant.
Key points:

1. **What it is**  A terminal UI/CLI that lets you chat with multiple AI models (Claude, GPT4, Gemini, local Ollama/LLM models) via a rich TUI or simple CLI commands.

2. **Quick Start**  Install with `pip install consoul`, set an API key, then run `consoul` for the TUI or use the SDK in Python.

3. **Features**
   * Beautiful TUI using Textual.
   * Multiprovider support (Claude, OpenAI, Gemini, Ollama, LlamaCpp).
   * Builtin AIpowered tools: file editing, code search, image analysis, bash execution, web search, URL fetching, etc.
   * Easy SDK integration for embedding AI into applications.

4. **CLI usage**  Quick examples for chat, ask, file attachment, etc.

5. **SDK**  Shows how to instantiate a `Consoul` object, enable tools, and make chat/ask calls.

6. **Documentation links**  Paths to full docs, configuration, TUI guide, code search guide, SDK guide.

7. **Configuration**  Sample `config.yaml` with profiles (default, local) showing model settings, tool policies, and conversation options.

8. **Security**  Highlights permission policies, approval workflows, audit logging, and command validation.

9. **Contributing & License**  MIT license and contributing guidelines.
